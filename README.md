Курсова робота виконана на тему розробки та аналізу надвеликої бази даних для екскурсійної компанії з використанням технологій Microsoft SQL Server. 
У роботі реалізовано повний цикл побудови аналітичної системи: від аналізу предметної області та проєктування операційної бази даних до створення сховища даних, ETL-процесів, OLAP-куба та аналітичних звітів.
У межах роботи було спроєктовано реляційну базу даних Excursions, яка відповідає бізнес-правилам екскурсійної діяльності та приведена до третьої нормальної форми. 
Для імітації реальних умов роботи системи згенеровано великий обсяг тестових даних (12 000 екскурсій, 500 автобусів, 400 водіїв, дані за 5 років) із використанням Redgate SQL Data Generator.
На основі операційної бази даних створено сховище даних Excursions_DWH зі зіркоподібною схемою, що включає таблиці фактів та вимірів. 
ETL-процеси реалізовано за допомогою SQL Server Integration Services (SSIS), що забезпечує коректне завантаження, трансформацію та очищення даних. 
Для аналітичної обробки даних побудовано багатовимірний OLAP-куб у SQL Server Analysis Services (SSAS) з налаштованими вимірами та ієрархіями.
Фінальним етапом роботи стало створення аналітичних звітів у Microsoft Excel з підключенням до OLAP-куба, зокрема: списку екскурсій за період, графіку використання автобусів, 
фінансового звіту та аналізу популярності маршрутів. Результати роботи демонструють можливість ефективного аналізу великих обсягів даних та можуть бути використані для 
підтримки управлінських рішень в реальній екскурсійній компанії.
